## [1.4.0] - 2026-06-14

### 🎉 初始版本

这是二次元聚合插件的首个正式版本，整合了多个独立插件的功能。

### ✨ 核心功能

#### 1. 壁纸功能
- `/壁纸 [分类]` - 获取随机或分类壁纸
- 支持分类：随机、二次元、风景、必应、手机
- 多图源备份机制
- 支持代理配置

#### 2. 二次元图片
- `/涩图 [关键词]` - 获取二次元图片
- `/涩图 r18 [关键词]` - 获取 R18 图片（需配置）
- 支持 Lolicon、Anosu、Dmoe 三个 API
- 关键词搜索（Lolicon API）
- 显示作品信息（标题、画师、PID）
- 冷却时间和每日限制
- R18 白名单机制

#### 3. 番剧搜索
- `/番剧 <关键词>` - 搜索番剧
- `/今日番剧 [星期]` - 查看放送表
- 基于 Bilibili 数据源
- 返回封面、评分、地区、标签、简介等
- 支持指定星期查看放送
- WBI 签名机制，绕过风控
- 自动获取和缓存 buvid3

#### 4. 以图搜索
- `/搜番图` - 以图搜番（trace.moe）
- `/搜图` - 以图搜图（SauceNAO）
- 支持三种发图方式
- 会话等待机制
- 相似度过滤
- SauceNAO API Key 支持

#### 5. 一言功能
- `/一言 [类型]` - 获取二次元语录
- 支持 12 种类型
- 可配置默认类型
- 显示语录出处

#### 6. Galgame 搜索
- `/gal <关键词>` - 聚合搜索
- `/galmoe <关键词>` - 萌娘百科搜索
- `/galkun <关键词>` - 鲲论坛搜索
- 返回作品信息、平台、语言等
- 支持话题搜索

### 🔧 技术特性

- **异步架构** - 使用 aiohttp 实现高性能异步请求
- **多源备份** - 壁纸、图片等功能支持多数据源自动切换
- **错误处理** - 完善的异常捕获和用户友好提示
- **缓存机制** - Bilibili buvid3 缓存，减少重复请求
- **会话等待** - 以图搜索支持会话等待功能
- **配置灵活** - 丰富的配置项，满足不同需求

### 📦 依赖

- aiohttp - 异步 HTTP 客户端

### 📝 配置项

- 壁纸配置：`wallpaper_timeout`、`wallpaper_proxy`
- 图片配置：`default_api`、`enable_r18`、`r18_whitelist_groups`、`daily_limit`、`cooldown_seconds`、`anime_proxy`
- 番剧配置：`bangumi_result_limit`、`bangumi_calendar_limit`、`bangumi_show_cover`
- 以图搜索：`trace_min_similarity`、`saucenao_api_key`、`saucenao_result_limit`、`image_search_timeout`
- Galgame 搜索：`result_limit`、`search_kun_topics`、`intro_max_length`
- 一言配置：`hitokoto_default_type`、`hitokoto_show_source`

## 版本说明

### 版本号规则

本项目遵循 [语义化版本](https://semver.org/lang/zh-CN/) 规范：

- **主版本号（Major）**：重大架构变更或不兼容的 API 修改
- **次版本号（Minor）**：新增功能，向下兼容
- **修订号（Patch）**：问题修复和小的改进，向下兼容

- ✨ **新增功能** - 新增的功能特性

- 📝 **文档更新** - 文档变更

- ❌ **移除** - 删除的功能
