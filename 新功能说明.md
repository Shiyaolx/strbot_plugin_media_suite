## 📚 漫画搜索功能

### 功能介绍
通过萌娘百科搜索漫画作品信息，支持日漫、国漫、韩漫等各类漫画。

### 使用命令
```
/漫画 <作品名>     # 搜索漫画
/搜漫画 <作品名>   # 别名
/manga <作品名>    # 英文别名
/comic <作品名>    # 英文别名
/漫画帮助          # 查看帮助
```

### 使用示例
```
用户: /漫画 进击的巨人

机器人:
📚 漫画搜索「进击的巨人」：
————————————
📖 进击的巨人
简介：《进击的巨人》（日语：進撃の巨人；英语：Attack on Titan）是由谏山创创作的一部漫画，于讲谈社《别册少年Magazine》2009年10月号上开始连载。並有動畫、游戏等衍生作品。
链接：https://moegirl.icu/进击的巨人

📑 其他相关结果：
· 巨人(进击的巨人)
  https://moegirl.icu/巨人(进击的巨人)
· 进击的oo
  https://moegirl.icu/进击的oo
```

### 功能特点
- ✅ 国内可直接访问，无需代理
- ✅ 返回详细的作品信息和简介
- ✅ 提供多个相关结果供选择
- ✅ 自动获取作品介绍

### 数据源
- **萌娘百科** (https://moegirl.icu)
- 稳定可靠，国内直连

## 😊 表情包功能

### 功能介绍
随机发送二次元表情包，适合在聊天中快速发送可爱的二次元表情。

### 使用命令
```
/表情              # 随机发送一个二次元表情
/表情包            # 别名
/emoji             # 英文别名
/来个表情          # 别名
/表情帮助          # 查看帮助
```

### 使用示例
```
用户: /表情

机器人: [发送一张随机的二次元表情图片]
```

### 功能特点
- ✅ 简单快速，一键发送
- ✅ 多个图源备用，高可用性
- ✅ 纯二次元风格表情
- ✅ 每次随机，不重复

### 图片源
1. **dmoe.cc** - 动漫图片库
2. **alcy.cc** - 萌图 API
3. **mtyqx.cn** - 随机动漫图
4. **btstu.cn** - 搏天 API（备用）

## 📝 配置说明

插件使用萌娘百科作为漫画搜索数据源，无需额外配置即可使用。

如需调整表情包或壁纸的超时时间，可在配置文件中修改：
```json
{
  "wallpaper_timeout": 20,
  "wallpaper_proxy": ""
}
```
